Results & Discussion

The use of micro-scale technology is an emerging field that presents useful applications in areas ranging from biomedical and biological engineering to environmental science. Throughout the course of the semester, our group was able to successfully program and execute a usable computational tool for the analysis, modeling, and extrapolation of trajectory data from superparamagnetic colloidal particles. These particles, once actuated with an external magnetic field, form MicroBots that can be used for minimally-invasive surgery within the tortuous environments of the human cardiovascular system.

In order to program a usable code for the real-world implementation of these MicroBots, the first aim of our project sought to perform image analysis on a series of MicroBot translation videos to determine the velocity of the robots with respect to a usable model equation. The first step of this goal entailed the creation of a series of functions that thresholded TIFF image files from each translation video, median filtered the MicroBot using its darker contrast against a white background, and interpreted each robot as a singular centroid over time. This first portion of our code reflects changes we made in response to peer feedback. Specifically, we noticed that by originally running the code utilizing the raw TIFF-format frames, the sheer amount of data took too long for the program to execute. In order to account for the relatively slower run time of this program, we implemented the use of centroid-oriented data that compacted the data extracted from each TIFF frame into a smaller set of X and Y coordinates. This reduced the overall run time of the image analysis portion of the code to only 3 to 5 minutes. Consequently, this step was successfully programmed as demonstrated by the plotted centroid paths of the MicroBots under magnetic field strengths of 20, 25, 30, and 35 mT. This result was crucial to the image analysis portion of the code because it demonstrated that the MicroBots, in vitro, could be tracked using image analysis software. Within the context of a real-world surgical operation, the capacity to track the MicroBots as they are magnetically actuated throughout the human body is incredibly important for visualizing the real-time location of each robot in relation to the site of a clot or thrombosis that needs to be targeted and lysed. The efficacy of our centroid path plot was also compared to the real video of a MicroBot moving under 20 mT field strength conditions. Given that both the centroid plot and the real MicroBot moved in a somewhat linear path across the screen, we were able to confirm the accuracy of our image analysis program.

After successfully determining both the coordinates of the MicroBot centroids and the paths of each MicroBot in each of the magnetic field strength videos, the program then used a series of functions to calculate the average velocity of the MicroBots in each video. This allowed us to determine the total velocities of the MicroBots, which were each in the range of roughly 6.829 microns per second. This was a result we expected because we knew the MicroBots themselves were roughly 4.5 microns in diameter, so a translational velocity in the same realm allowed us to determine the velocity function was successful.
